EXAMPLE CARRIED OUT IN VISUALSTUDIO 2017
The solution is divided into three projects to show the use of the library in X64 and in X32 (Cdecl and StdCall)

Functional program to test the manipulation of shared memories in both 
32-bit and 64-bit architecture.

*** X86_32
In a 32bit windows, the libraries located in "Libraries/32Bits" must be located inside the "C:/Windows/System32" folder.

*** X86_64
In a 64bit windows, the libraries located in "Libraries/32Bits" must be located inside the folder "C:/Windows/SysWOW64", and the libraries located in "Libraries/64Bits" must be located inside the "C:/Windows/System32" folder.
